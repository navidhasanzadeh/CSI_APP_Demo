Actions, Scripts, and Repetitions

Action profiles define the gesture set shown during an experiment. Each action can point to a video or script that guides the participant.

What you can configure

  • Action names and their associated media files.
  • Order of actions using drag-and-drop rows.
  • Number of repetitions per action (from the experiment profile).
  • Optional scripts for automation or integration.

Example

Add actions "left_right", "up_down", and "push_pull" with matching video previews. If you want to add an automation step, point the script field to a folder like scripts/default and select a script type.
