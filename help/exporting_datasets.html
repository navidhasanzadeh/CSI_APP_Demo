Exporting Datasets

Exporting packages your experiment PCAP data into a shareable dataset file for offline analysis and model training.

Where to export

  1. Open the PCAP reader.
  2. Select the experiment folder you want to export.
  3. Click Export Dataset to choose a destination file.

Example

After finishing a session, open the PCAP reader, select the experiment folder, and export to exports/session_2024_11_05.pkl. This file can then be loaded in Python for feature extraction and model training.
