Configuration Window Overview

The configuration window is the main place to prepare your experiment. It is split into tabs (Participant, Experiment, Environment, Cameras, Voice, UI, Time, Wi-Fi, Actions) so you can manage one type of configuration at a time.

Typical workflow

  1. Select or create a participant profile.
  2. Pick an experiment profile that defines timing and save locations.
  3. Assign action profiles and scripts that match your protocol.
  4. Configure Wi-Fi/PCAP capture settings.
  5. Adjust camera, voice, UI, and time settings as needed.
  6. Click Start Experiment when everything looks good.

Use the Help button at the top of the window to browse detailed guides and examples for each section.
