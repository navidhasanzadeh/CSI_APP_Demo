PCAP Reader & Plotting

The PCAP reader lets you browse captured experiments, inspect CSI packets, and plot signal metrics over time.

Exported dataset pickles

Use Export Dataset in the reader to save a pickled dictionary that contains CSI tensors plus metadata. The dictionary keys are (participant, sniffer, environment, trial, action) tuples and each entry stores csi, timestamps, and summary metadata. CSI is shaped as [time, subcarrier, rx, tx] (or [time, subcarrier, rx] for 1SS captures).

Example: load a pickle and inspect CSI

import pickle

with open("/path/to/exported_dataset.pkl", "rb") as handle:
    data = pickle.load(handle)

key = ("Navid", "Sniffer 6", "demo1", "9", "Push pull")
entry = data[key]

print(entry["csi"].shape)
print(entry["timestamps"][:5])

Example: plot a sample signal

import matplotlib.pyplot as plt
import numpy as np

csi = entry["csi"]  # [time, subcarrier, rx, tx]
plt.figure()
plt.plot(np.abs(csi[:, 23, 0, 0]))
plt.title("Subcarrier 23 amplitude (Rx0, Tx0)")
plt.xlabel("Packet index")
plt.ylabel("Amplitude")
plt.show()

Example: plot CSI ratio angle

import matplotlib.pyplot as plt
import numpy as np

ratio = entry["csi"][:, 23, 0, 0] / entry["csi"][:, 23, 0, 1]
plt.figure()
plt.plot(np.angle(ratio))
plt.title("Angle of CSI ratio (Subcarrier 23, Rx0, Tx0/Tx1)")
plt.xlabel("Packet index")
plt.ylabel("Angle (radians)")
plt.show()

Typical flow

  1. Open the PCAP reader window.
  2. Select an experiment folder that contains PCAP files.
  3. Choose an action or segment to visualize.
  4. Use the plot controls to adjust subcarriers, antenna pairs, and time ranges.

Settings worth checking

  • Subcarrier range: start with a handful of stable subcarriers before expanding to all.
  • Rx/Tx pairs: confirm the antenna pair matches your hardware setup.
  • Time window: zoom into a single action window for cleaner plots.
  • Amplitude vs phase: phase plots may need unwrapping or smoothing.

Common issues

  • Empty plots: verify the correct action folder and that packets were captured.
  • Jagged phase: check for dropped packets or apply phase unwrapping.
  • Unexpected shape: 1SS captures omit the Tx axis; adjust indexing accordingly.

Example

If you suspect a noisy capture, load the action "left_right" and plot a few subcarriers. Look for sudden amplitude spikes or dropouts that indicate packet loss or interference.
