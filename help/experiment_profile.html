Experiment Profile

The experiment profile controls timing and storage for each capture session.

Key timing parameters

ParameterMeaning
Beginning baselineIdle recording time before the first action.
Between actions baselineIdle recording time between actions.
Ending baselineIdle recording time after the final action.
Action timeDuration of the guided action.
Stop timePause before advancing to the next action.
Preview pauseDelay before showing the action preview.
Repetition countNumber of times each action is repeated.

Example

For a 10-action experiment, set Beginning baseline to 15 seconds, Action time to 4 seconds, and Between actions baseline to 10 seconds. This produces a consistent buffer around each gesture so the PCAP and camera data have context for labeling.
