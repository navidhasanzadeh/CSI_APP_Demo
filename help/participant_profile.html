Participant Profile

The participant profile captures who is taking part in the experiment. It includes demographic details and optional secondary participant information.

Key parameters

  • Name and Participant ID for traceability.
  • Age group, gender, height, and weight for metadata.
  • Dominant hand to explain directional gestures.
  • Description for notes (e.g., "left-handed, wrist brace").

Example

Use Name: "Participant_01234", Age group: 18-30, Dominant hand: right, and add a description such as "experienced with gesture set A".
